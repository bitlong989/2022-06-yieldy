Currently if you stake for FOX, and then want to unstake your fox, you have a 7-14 day window in which that FOX will become claimable due to tokemak's cycles. Alternatively, if the staking token and the tAsset (from tokemak) have a curve pool we can allow the user to swap there for immediate liquidity instead.
